# Setup Streaming Analytics & IOT

# Sign in
```
az login
```

# Init terraform
```
terraform init
```

# Run Terraform plan
```
terraform plan
```

# Run Terraform apply
```
terraform apply
```

# Cleanup Demo
```
terraform destroy
az group delete --name demo-stream-iot --no-wait
```
